# 요구사항
---

- 면접 복기 입력을 받아야함
    - 간단하게 입력할 수 있는 선택지가 있으면 좋겠음
        - 심사위원들이 한 질문들
        - 대답 못한 부분
        - 대답을 잘한 부분
- 멘탈 관리, 이때 느낀 감정, 멘탈 케어
    - 멘탈 준비 후기(인데놀, 청심환, 마인드 컨트롤)
    - 언제 먹었는데 언제 효과가 있더라, 또는 어떻게 안맞더라
- 내 면접 관리(기록)
    - 통계 등
    - 반복된 실수
    - 나에게 자주 나오는 질문
- 면접 피드백 리포트 생성
    - 다음 면접 전략을 세워야 함
    - 부족한 부분을 알 수 있어야 함
    - **멘탈 관리 조언(행동 추천-미션 기반)**
- 기업 공고 입력 시 기반으로 차후 플젝 디벨롭이나 역량 향상 미션 제공
